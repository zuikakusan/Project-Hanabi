## 3. 组织表

### 3.1 大组织：放学后

| 字段 | 内容 |
|---|---|
| 名称 | 放学后 |
| 类型 | 本地孩子自发组成的松散秘密组织 |
| 公开程度 | 大人不知道，孩子之间通过传闻和暗号传播 |
| 共同目标 | 看看这个世界的真相 |
| 组织气质 | 即兴、松散、认真、带有儿童式仪式感 |
| Demo 呈现 | 只在结尾通过名册、徽章和纸条出现 |

组织层级：

```text
放学后
  -> 纺织厂分会
  -> 钢厂分会
  -> 老街分会
  -> 农村分会
  -> 城中村分会
```

## 6. 敌人表

### 6.1 Demo 敌人

| ID | 名称 | 行为 | 弱点 | 用途 |
|---|---|---|---|---|
| enemy_paper_ball | 纸团怪 | 慢速接近，近身攻击 | 摔炮、魔术棒 | 基础教学 |
| enemy_shadow_kid | 影子小孩 | 快速移动，绕侧面 | 擦炮预判 | 逼玩家移动和控场 |
| enemy_guard_echo | 保安幻影 | 对声音敏感，追击玩家 | 绕行、低噪音武器 | 噪音系统教学 |
| boss_duty_monitor | 值日生 | 召唤纸团怪，远程黑板擦 | 窜天猴打断 | Demo Boss |

表现原则：

- 敌人优先幻想化，不做真实儿童作为战斗击败对象。
- 受击反馈使用火花、纸屑、影子散开、勇气震动。
- 避免血腥、真实伤害和死亡表现。

## 7. 小游戏表

### 7.1 小刀一把

| 字段 | 内容 |
|---|---|
| 类型 | 猜拳 + 回合博弈 |
| 玩家数 | 2 |
| 核心资源 | 位置、道具、面子、防守状态 |
| 胜利条件 | 让对方出局 |
| Demo 对手 | 王小虎 |
| 奖励 | 后巷线索、面子、王小虎态度变化 |
| 失败兜底 | 可重试，或由阿杰给出备用线索 |

行动表：

| 行动 | 条件 | 效果 |
|---|---|---|
| 拿道具 | 自己阵地 | 获得一级纸刀 |
| 前进 | 未到对方阵地 | 向前移动一格 |
| 出击 | 在对方阵地且有道具 | 对方出局 |
| 抢道具 | 对方在自己阵地且持有道具 | 概率夺取 |
| 防守 | 任意位置 | 提高下次防守成功率 |
| 嘴硬 | 任意位置 | 提升面子，增加失败惩罚 |

## 9. 属性表

正式版建议使用儿童化属性。

| 属性 | 作用 |
|---|---|
| 胆量 | 进入鬼屋、面对敌人、特殊对话 |
| 机灵 | 解谜、识破谎言、发现隐藏线索 |
