最后更新：2026-06-16  
状态：MVP 减法方案

# 一、概述

**游戏名称**：《千禧年战纪》（备选：《千禧年传说》）

**一句话概括**：

回到千禧年的小县城，白天在小学门口拍画片、打弹珠，课间玩小刀一把，中午放学去小卖铺围观松鼠大作战，下午放学后拿着窜天猴和魔术弹，和伙伴们在鬼屋和秘境里，把童年的幻想打成真的。

**参考系**：

|   |   |
|---|---|
|游戏|借鉴什么|
|《女神异闻录5》|日常/非日常双重生活结构|
|《放学后少年》|日常小游戏设计思路（弹珠、圆牌、拍画片）|
|《地痞街区》|俯视角战斗的混乱感和自由度|
|《逃离鸭科夫》|据点-地图往返的循环节奏|

**游戏类型：3D俯视角第三人称射击、开放世界探索**

**目标体验**：

- 日常线（60% 时间）：怀旧温馨——弹弹珠、打圆牌、在学校门口闲逛
- 冒险线（40% 时间）：紧张刺激——秘境探索、烟花战斗、天黑前必须回家（夕阳，作为游戏的主色调）
- 两个情绪互相衬托：温馨让刺激更刺激，刺激让温馨更珍贵

**核心循环**：

|   |   |   |
|---|---|---|
|层级|时长|内容|
|日常微循环|~15分钟|上学的日常、和小伙伴互动 → 玩小游戏（弹珠/圆牌/小刀一把）→ 获取资源/情报→前往商店、购买道具|
|冒险微循环|~10 秒|发现目标 → 点燃烟花 → 瞄准发射 → 反馈 → 立刻转移|
|冒险中循环|~15 分钟|进入秘境 → 探索+战斗 → 搜刮奖励 → 在天黑前撤离回据点|
|大局循环|无限制|不同区域探索 → 推进主线秘密 → 解锁新能力和新区域|

## 二、游戏背景设定

这个世界有两面——大人们只看得见表面的那一面。但每个千禧年的小孩都知道，在小区假山后面、在学校废弃的传达室里、在家属院废弃二层楼，藏着另一个世界。

### 2.1 世界观：千禧年的双面小城

**安城**。一座典型的中国中部工业小城，时间停在 2008 年。

白天的安城是灰色的——纺织厂的烟囱吐着白烟，自行车铃铛在街头巷尾叮当作响，小学生在放学的队伍里背着手走路。大人们的世界井然有序：上班、下班、买菜、看新闻联播、催孩子写作业。在大人眼里，安城没有什么秘密。

但孩子的眼睛不一样。

孩子知道纺织厂三号车间的灯半夜会自己亮，知道学校后门那条"死胡同"往里走七步右转会多出一条街，知道每个小区的假山都有一条大人钻不进去的缝隙。孩子还知道——电视和课本上说的"科学"，没办法解释为什么四号家属院的地下室每到星期三就传出算盘声，而那个地下室十年前就封死了。

**这就是安城的另一面**：一座隐藏在寻常街巷之下的、只在孩子眼中显现的"秘境之城"。那些被大人称之为"小孩子瞎想"的地方，是真实存在的。而探索这些地方的人，就是——

### 2.2 神秘组织：「小刀一把」

**"小刀一把"** 是一个由安城小孩自发组成的秘密社团。它没有创始人、没有总部、没有成立日期。没人知道它是什么时候开始的，就像没人知道第一个把弹珠放进圈里算"入会仪式"的小孩是谁。

**组织信条只有一条**：_"看看这个世界的真相。"_

**组织架构**（即兴的、松散但真实的）：

```
                    「放学后」
                        │
        ┌───────┬───────┼───────┬───────┐
        │       │       │       │       │
    纺织厂分会  钢厂分会  农村分会  老街分会  （更多……）
        │
   ┌────┴────┐
   │         │
 五年级组   六年级组
   │
小陈、阿杰、胖子、小美……
```

- **大组织层面**："放学后"只是一个共同身份，就像 RPG 里的"冒险者协会"——你有了这个身份，就能在组织据点之间安全通行，能看懂暗号，能进组织的秘密商店
- **分会层面**：每个分会根据地盘划分（纺织厂家属院的孩子是一个分会，钢厂家属院的是另一个），分会之间可能是盟友（一起打隔壁小区），也可能是对头（抢秘境资源）
- **小队层面**：三五好友组成的最小行动单位，就是你日常一起玩的那几个同学

**加入方式**：每个孩子的入会方式都不一样。有人是赢了十局弹珠后 NPC 递给他的暗号纸条，有人是在家里阁楼上翻出了哥哥留下的旧地图，有人是被"鬼屋"里的声音叫进去的。而小陈的方式是——

### 2.3 主角故事：小陈的入场

**陈小宇**，大家叫他小陈，11 岁，纺织厂子弟小学五年级二班。

他的世界很简单：

- 家住纺织厂家属院 3 号楼 402，窗户对着废品站的院子
- 爸爸是纺织厂修机器的，手上永远有洗不掉的机油味
- 妈妈在小学门口开了个小卖部，卖辣条、贴纸和玻璃弹珠
- 成绩中不溜，语文比数学好，最怕写《我的理想》这类作文——他确实没什么理想

小陈不是那种领头打架的孩子，也不是那种被欺负不敢吭声的孩子。他是那种**放学后会故意绕远路回家的人**——不为别的，就是想看看那条小巷子今天有没有什么不一样。

一个普通的星期二，下午放学后。小陈和往常一样绕路，经过学校后门那间废弃的传达室。传达室的门永远锁着，窗户里面黑漆漆的，玻璃上糊着旧报纸。但今天不一样——**窗户裂了一条缝，里面好像有人在看着他。**

他把手伸进裂缝，摸到了一本册子。

封面已经被翻得起了毛边，上面用圆珠笔歪歪扭扭地写着四个字：**"会员名册"**。里面密密麻麻记着名字、班级、和一个奇怪的编号。他翻了翻，在最新的一页，看到了自己班里阿杰的名字。

阿杰？那个连体育课都不敢逃的阿杰？

小陈往后翻，发现最后一页是空白的。最下面写着：

_"下一个找到这本册子的人，把名字写在这里。然后去_**_xx小区家属院42号楼_**_。记住——太阳下山之前。"_

42号楼？这是在哪？有这个地方？

小陈站在传达室门口，手里捏着册子。天边的云开始变红。

他看了一眼表：下午 4 点 47 分。

他把名字写了上去。

### 2.5 叙事基调总纲

这不是一个"拯救世界"的故事。这是一个"在千禧年的中国小城里，放学以后，天黑之前，用烟花和想象力，探索一个只有小孩才能看见的世界"的故事。

**叙事原则**：

- **大人的世界是模糊的背景**——父母、老师、保安大叔，他们存在，但他们不是反派。他们只是"看不见"另一个世界的人
- **小孩之间的关系比"正义 vs 邪恶"更复杂**——今天是敌人，明天可能因为一颗弹珠变成朋友；不同分会之间的矛盾是真实的，但没有一个小孩是"纯粹的坏人"
- **秘境不是地牢，是童年想象的具象化**——废弃车间在大人眼里是破房子，在小孩眼里它是敌人的要塞、是藏着秘密的基地、是通往另一个世界的入口
- **怀旧不是道具**——不是靠贴几张千禧年海报来怀旧。怀旧是你在游戏里听到远处传来的《还珠格格》片尾曲，是放学后校门口辣条摊的味道（视觉化），是你妈在楼道里喊你回家吃饭的那个声音
- **勇者斗恶龙的三元组**——勇者斗恶龙，勇者最后是要打败魔王，这就对吗？（邪神隐藏在暗处）
- **后日常系/新日常系**：温馨的日常背后，有一条玩家无法察觉的暗线在暗流涌动。

### 2.6 场景1 老孟的神秘商店

在小学门口斜对面的第三个小卖铺，它是整个小学周围最大的小卖铺，货也很多，摆满了整个店面——都是孩子们喜欢的小玩意。老板是个中年男性，不过还算精神、胡子剃的很干净、能看出锋利的棱角，被孩子们称为老孟。

## 三、最小demo

原则：一个主角、一个区域、三个烟花、一个秘境、一条任务线。  
目标：2 个月内能在 Godot 里跑通一段 10 分钟的 gameplay。

### 3.1 主角（1 个 MVP，其余砍掉）

**小陈**，11 岁，五年级，住在**纺织厂家属院**。

- 父亲是纺织厂工人，母亲在小学门口开小卖部
- 性格：好奇心强，有点怂但关键时刻敢上
- 初始区域：纺织厂家属院小区 + 纺织厂子弟小学+
- 接触神秘组织的方式：在小学后门的废弃传达室里发现了一本"会员名册"

砍掉：其他所有主角。做完小陈的完整故事线再考虑加人。

### 3.2 最小地图（3 个地点）

```
        （家） ←──步行──→ 纺织厂第一小学（日常区）
                                      │
                                      │ 放学后解锁
                                      ↓
                                 家属院小区（秘境入口/探索地图/日常区域）
                                      │
                                      ↓
                             家属院废弃二层楼（42号楼）（第一个秘境/冒险区，后作为秘密基地）
```

|   |   |   |
|---|---|---|
|地点|大小|功能|
|**家（家属院 3 号楼）**|1-2 个房间|起床/回家存档/和父母对话/触发任务|
|**小学**|校门口 + 操场 + 教学楼走廊|日常线主场：小游戏、和 NPC 对话、获取情报、小卖部购买道具|
|**家属院废弃二层楼**|3-5 个房间|冒险线主场：烟花战斗、解谜、探索|

砍掉：农村、其他小区、公交系统、鬼屋、神秘商店、工会交易。MVP 只需要一个秘境。

### 3.3 日常线（3 个小游戏 + 对话）

|   |   |   |
|---|---|---|
|小游戏|规则|奖励|
|**弹弹珠**|俯视角，瞄准→弹射，打进圈得分|零花钱、情报|
|**拍圆牌**|时机判断，看准拍下翻对方的牌|圆牌|
|**盗版宝可梦**|剪刀石头布|宝可梦卡牌|
|**气功师**|另类剪刀石头布||
|**小刀一把**|石头剪刀布 + 回合博弈|和特定 NPC 建立关系，解锁秘境情报|
|**FC游戏机**|||
|**小卖铺抽奖**|||

日常行动范围：小学门口（小卖部、弹珠摊）、操场（圆牌场）、教学楼走廊（同学对话）。

砍掉：偷菜、生火、其他所有 mini-game。三个足够支撑日常线。

### 3.4 冒险线（1 个秘境 + 3 种烟花）

**秘境： 家属院废弃二层楼**

- 3-5 个房间，每个房间有小规模战斗或简单解谜
- 敌人：其他小孩帮派成员（PvE）+ 偶尔路过的保安大叔（精英敌人，被发现=强制撤离）
- 天黑倒计时：进入秘境后开始计时，必须在倒计时结束前撤离
- 奖励：稀有烟花、故事碎片、解锁新能力

**4 种 demo 烟花武器：**

|   |   |   |
|---|---|---|
|烟花|类型|效果|
|擦炮|投掷/aoe|抛物线投掷、延迟中范围爆炸、伤害中等。|
|窜天猴|直线远程|瞄准方向直线飞出，命中爆炸，声音大容易暴露位置|
|摔炮|投掷/AOE|抛物线投掷，落地小范围爆炸，安静但伤害低|
|魔术棒|点燃后从顶端持续发射弹药|点燃后，每过1s就可以发射一枚子弹，造成小伤害，一个魔术棒发射持续一分钟。|

砍掉：鞭炮、二踢脚、玩具枪、木棒、双节棍、仿真军刀。3 种烟花已经能做出战术差异。

### 3.5 冒险线触发条件

日常 → 冒险的触发：**每天放学后**（游戏内时间 16:00），玩家可以选择：

- 直接回家（安全，推进日常线）
- 前往秘境（冒险，消耗天黑前的时间窗口）
- 闲逛（随机触发小事件）

关键：天黑前必须回家。超过时间未归 = 被父母发现 = 道具没收 + 第二天禁足（无法进入冒险线一天）。

### 3.6 MVP 需要搭建的系统

|   |   |   |
|---|---|---|
|系统|优先级|说明|
|俯视角角色移动|P0|大地图和秘境的基础|
|对话系统|P0|NPC 对话、任务触发|
|昼夜时间系统|P0|天亮→天黑，驱动日常/冒险切换|
|烟花战斗（3 种）|P0|瞄准 + 发射 + 反馈|
|弹珠小游戏|P1|日常线核心交互|
|圆牌小游戏|P1|日常线核心交互|
|道具背包|P1|烟花和收集品管理|
|秘境房间切换|P1|废弃车间的房间推进|
|小刀一把|P2|可以先用简单版本，后期迭代|
|NPC 好感/关系|P2|MVP 阶段用对话树代替|
|存档系统|P2|MVP 可以先做自动存档|

## 四、砍掉清单（记着，以后再加）

- ❌ 多主角 → MVP 只有小陈
- ❌ 农村、其他小区、公交系统 → MVP 只有纺织厂家属院周边
- ❌ 鬼屋/超自然秘境 → MVP 用废弃车间（现实向秘境，降低美术成本）
- ❌ 工会/交易系统 → MVP 不需要
- ❌ PvP → MVP 只做 PvE
- ❌ 多区域组织 → MVP 只有一个敌对小孩帮派
- ❌ 偷菜、生火、大量 mini-game → MVP 3 个小游戏
- ❌ 5+ 种烟花 → MVP 3 种
- ❌ 多种武器（玩具枪、木棒等）→ MVP 只要烟花
- ❌ 黑夜解锁 → MVP 暂不做（天黑=强制回家就够了）

## 五、10 分钟 Gameplay 脚本（MVP 目标）

```
【日常线 - 放学】
16:00，下课铃响。小陈走出教学楼。
→ 玩家自由走动，和操场上的同学对话
→ 一个同学说："听说后门传达室有鬼，你敢去吗？"

【日常线 - 小游戏】
→ 玩家去校门口弹珠摊，和 NPC 打一局弹珠，赢了
→ NPC 透露："那个传达室里好像有个本子，写满了名字……"

【触发冒险】
→ 玩家走到后门传达室，发现一本破旧的"会员名册"
→ 上面写着一个地址：纺织厂废弃车间
→ 屏幕提示：天黑还有 2 小时

【冒险线 - 秘境探索】
→ 进入废弃车间，3 个房间
→ 第 1 个房间：遭遇敌对小孩帮派成员，用窜天猴远程解决
→ 第 2 个房间：解谜（推箱子/找钥匙），用摔炮清除障碍
→ 第 3 个房间：Boss 战（帮派老大），用魔术弹贴身战斗
→ 胜利，获得稀有道具 + 组织信物

【返回日常】
→ 天黑前赶回家
→ 母亲在门口："又去哪疯了？洗手吃饭！"
→ 晚上在房间里翻阅今天的收获
→ 存档，结束
```

我想的一个最小游玩demo是一个不同孩子之间的小游戏：被称之为小刀一把 他是这样玩的：两个玩家进行石头剪刀布，赢得玩家获得一次行动机会，设定一开始两个玩家在自己的家中，初始等级为1，赢了就可以获得1级的装备小刀，然后再进行石头剪刀布，赢了就可以前往对方家里，继续石头剪刀布，赢了就可以击杀对方，这里之前输的人也有翻盘机会就是对方来到自己家的时候，你可以抢夺他的武器然后击杀对方，当你杀死对方一次的时候，你就可以提升等级，重新开始游戏，你可以获得更好的道具，当然对方也可以抢夺你的武器，低等级的玩家仍然具有翻盘机会  
我想做成类似小游戏合集、这个只是其中一个小游戏，核心玩法是大地图探索，然后进入秘境使用烟花爆竹进行战斗，做成p5那种模拟小学生生活的游戏，也参考放学后少年这个游戏的小游戏设计
