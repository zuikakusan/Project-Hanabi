# 文本风格手册

## 1. 总体叙事原则

《千禧年战纪》的文本要写出两件事：

1. 千禧年前后中国小学生日常的具体质感。
2. 熟悉世界背后逐渐显露的异常。

它不是宏大史诗，也不是纯恐怖故事。它更像一次放学后的冒险：孩子们把小卖部、楼道、操场和废弃二层楼当成真正的世界边界，而黄昏之后，这个边界真的开始松动。

## 2. 叙事基调

| 层级 | 基调 | 写法 |
|---|---|---|
| 日常 | 温馨、具体、轻微好笑 | 用细节写时代感和儿童关系 |
| 小游戏 | 较真、讲规则、在意面子 | 把儿童游戏写得像江湖决斗 |
| 秘境 | 梦核、压迫、不合理 | 熟悉场景发生轻微错位 |
| 组织 | 神秘、松散、儿童式认真 | 像秘密基地，不像正规机构 |
| 真相 | 克制、暗流、后日常 | 不急着解释，先让异常留下痕迹 |

## 3. 文本视角

### 3.1 儿童视角

文本应该优先从孩子能理解的方式描述世界。

推荐：

- “那扇门今天好像比昨天矮一点。”
- “王小虎说话声音很大，因为旁边有人看着。”
- “你知道这不是普通的楼道。普通楼道不会有这么多层。”

避免：

- “该区域呈现非欧几里得空间结构。”
- “此处为异常维度裂隙。”
- “NPC 受到社会性羞辱机制影响。”

设计原则：玩家可以感觉到背后有系统，但角色不该用成人化术语解释系统。

### 3.2 大人视角

大人不是反派，更多是背景、限制和现实重力。

大人常说：

- “写完作业再出去。”
- “别去那种破地方。”
- “小孩子懂什么。”
- “天黑之前回来。”
- “你是不是又买这些没用的东西？”

大人的话语应让玩家感到生活真实，而不是简单压迫。

### 3.3 秘境视角

秘境文本要克制。

推荐写法：

- 先写熟悉的东西。
- 再写一个不合理的细节。
- 不立刻解释。

例：

> 楼道里的声控灯亮了。  
> 灯下面贴着一张值日表。  
> 你的名字在上面，日期是明天。

## 4. 对话风格

### 4.1 小学生对话

关键词：

- 嘴硬
- 较真
- 含糊
- 爱用传闻压人
- 在意围观者
- 不直接承认害怕

例：

```text
王小虎：你敢去 42号楼？
小陈：谁不敢？
王小虎：那你现在去。
小陈：现在不行，我书包还在教室。
王小虎：怂就说怂。
小陈：玩一把。谁输了谁才怂。
```

### 4.2 神秘 NPC 对话

神秘 NPC 不要谜语太满。最好是在普通话里夹一点不对劲。

例：

```text
老孟：摔炮拿一盒。别嫌声音小，那边有时候声音小才好。
小陈：哪边？
老孟：你不是已经知道了吗？
```

### 4.3 父母对话

父母对话要具体、生活化。

例：

```text
妈妈：桌上两块钱，别全买辣条。
妈妈：放学早点回来。你爸今天夜班，没人下楼找你。
小陈：知道了。
妈妈：你每次都知道。
```

## 5. 内心声音系统

内心声音可以参考《极乐迪斯科》，但必须儿童化。它不是成年人的哲学独白，而是孩子脑内被放大的本能、面子、害怕、幻想和作业压力。

### 5.1 内心能力表

| 能力 | 语气 | 用途 |
|---|---|---|
| 胆量 | 硬撑、鼓劲 | 面对鬼屋、威胁、挑战 |
| 机灵 | 抓漏洞、看破绽 | 解谜、识破谎言 |
| 面子 | 在意别人怎么看 | 小游戏、冲突、选择 |
| 幻想力 | 把异常看成真实 | 秘境、隐藏物、怪事 |
| 手巧 | 注意结构和材料 | 道具制作、机关 |
| 记仇 | 翻旧账 | 复仇、竞争、支线 |
| 作业压力 | 现实拉扯 | 时间限制、家庭压力 |

### 5.2 内心声音样例

胆量：

> 不能退。退了王小虎能说到下学期。

机灵：

> 他刚才说“又”。这说明他不是第一次去后巷。

面子：

> 旁边至少有六个人在看。六个人，差不多就是全班都知道了。

幻想力：

> 这不是普通的门。普通的门不会等你。

手巧：

> 这根引线还能接。接得不好会炸手，接得好就会很响。

作业压力：

> 你还有三道应用题没写。它们现在安静地躺在书包里，但不会一直安静。

## 6. 中式梦核写法

中式梦核不是把场景写成恐怖屋，而是把玩家熟悉的生活场景轻轻扭歪。

### 6.1 推荐元素

- 学校走廊突然变长
- 小卖部老板永远背对玩家
- 家里的门打开后变成学校厕所
- 操场中央出现没人承认存在的沙坑
- 课本插图里的孩子少了一个
- 电视播放明天才会发生的事
- 黄昏后楼梯通向不存在的楼层
- 放学铃响后，学校里没有任何人
- 小卖部后门通向神秘街道
- 楼道声控灯只在玩家不说话时亮

### 6.2 写作模板

```text
普通细节 -> 不合理细节 -> 角色反应 -> 留下问题
```

例：

> 走廊尽头还是那个拖把池，瓷砖缺了一块。  
> 但你记得它应该在一楼。  
> 王小虎没有说话。他刚才还挺能说的。  
> 你们同时看见拖把池下面有一行粉笔字：今天谁值日？

## 7. 儿童冲突表达规范

由于角色主要是儿童，文本应避免写实未成年人伤害。

### 7.1 推荐词

- 出局
- 被判负
- 被击退
- 失去勇气
- 火花熄灭
- 被梦境吞没
- 回到现实
- 怂了
- 跑了
- 被吓住

### 7.2 避免词

- 杀死
- 处决
- 致命伤
- 血流
- 尸体
- 真实刀伤
- 未成年人之间的写实暴力

### 7.3 小刀一把表达

“小刀一把”建议作为抽象规则游戏表现。

| 原始概念 | 文本表达 |
|---|---|
| 杀死对方 | 让对方出局 |
| 小刀 | 纸刀、木刀、规则道具 |
| 对方家 | 对方阵地 |
| 抢武器反杀 | 抢到道具后反击 |
| 升级 | 段位提升、面子提升 |

## 8. 任务文本样例

### 8.1 任务名

推荐任务名：

- 42号楼的名字
- 放学后别回头
- 谁把门画在墙上
- 老孟今天没开灯
- 值日表上没有的人
- 第七条走廊
- 星期三的算盘声

### 8.2 Demo 主任务目标文本

| 阶段 | UI 目标文本 |
|---|---|
| 出门 | 背上书包，去学校。 |
| 找王小虎 | 去小卖部门口找王小虎。 |
| 小游戏 | 赢下“小刀一把”，问出 42号楼的事。 |
| 后巷 | 放学后去学校后巷看看。 |
| 神秘商店 | 走进多出来的那条街。 |
| 备战 | 从老孟那里拿到爆竹。 |
| 秘境 | 在天黑前调查 42号楼。 |
| Boss | 让“值日生”出局。 |
| 结尾 | 查看储物柜里的旧名册。 |

## 9. 场景调查文本样例

### 9.1 小陈家

电视机：

> 电视里正在播广告。声音很亮，画面有点糊。  
> 你不记得这个广告是卖什么的，但你会唱最后那句。

书包：

> 书包很沉。  
> 里面有课本、作业本、半块橡皮，还有一些你不打算让妈妈看见的东西。

挂历：

> 今天被红笔圈了一下。  
> 你不知道是谁圈的。家里人一般只圈发工资的日子。

### 9.2 学校后巷

铁门：

> 铁门锁着。  
> 门缝后面不是操场外墙，而是一条你没见过的街。

粉笔字：

> 墙上写着：别告诉大人。  
> 字很歪，但每一笔都很用力。

### 9.3 42号楼

声控灯：

> 你跺了一下脚。  
> 灯没有亮。  
> 你屏住呼吸，它亮了。

值日表：

> 值日表已经发黄。  
> 最后一栏写着你的名字。  
> 日期是明天。

黑板：

> 黑板上写着“今天谁值日”。  
> 粉笔灰落在讲台上，像很久没人擦，又像刚刚才写完。

## 10. 组织文本样例

入会纸条：

```text
你已经看见了。

放学后，到老地方。
别告诉大人。
```

组织信条：

> 看看这个世界的真相。

儿童暗号：

```text
问：今天星期几？
答：放学以后不算。
```

## 11. 命名规范

### 11.1 区域命名

推荐：

- 42号楼
- 学校后巷
- 消失的第七条走廊
- 四号院地下室
- 老街的半个店铺
- 南山防空洞

原则：名字要像本地孩子口口相传的地名，不像系统术语。

### 11.2 道具命名

推荐：

- 摔炮
- 擦炮
- 魔术棒
- 窜天猴
- 圆牌徽章
- 不会用完的粉笔
- 黑色橡皮
- 旧名册碎页

原则：普通物品可以有异常效果，但名字先保持普通。

## 12. 文本禁区

暂时避免：

- 过早解释秘境本质
- 大量成人哲学术语
- 把儿童组织写成正规军事组织
- 把怀旧写成品牌和道具堆砌
- 把恐怖写成血腥惊吓
- 把小学生对抗写成真实致命暴力
