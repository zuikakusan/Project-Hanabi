# 最小 Demo 氛围与文本替换方案

## 1. 文档目的

这份文档用于在“废弃小屋 Demo”功能跑通后，为它替换一层更接近《千禧年战纪》的氛围、文本和视觉方向。

当前实现者应优先完成 [[10 最小Demo实现交付说明]] 中的功能闭环。本方案不要求第一版立刻实现，但策划可以提前准备好，等功能通过验收后再进入 v0.2 氛围版。

## 2. 氛围版目标

功能版目标：

```text
纸条 -> 门 B -> 房间 A -> 柜子钥匙 -> 门 A -> 户外终点
```

氛围版目标：

```text
普通废弃小屋 -> 不太对劲的纸条 -> 被锁住的出口 -> 旧柜子里的钥匙 -> 门外异常的黄昏
```

氛围版不改变主流程，只改变：

- 文本语气
- 物件命名
- 场景细节
- 光线气质
- 终点提示
- 与《千禧年战纪》世界观的轻微连接

## 3. 氛围定位

### 3.1 关键词

- 千禧年初
- 小学生视角
- 废弃小屋
- 黄昏
- 灰尘
- 旧木门
- 生锈钥匙
- 发黄纸条
- 不该出现的出口
- 荧光绿
- 中式梦核
- 轻微不安，不做强恐怖

### 3.1.1 荧光绿定位

荧光绿是本 Demo 氛围版的重要梦核点缀色。

它不作为大面积主色使用，而是作为“异常信号色”出现：当普通废弃小屋里突然出现一点不自然的荧光绿，玩家会意识到这个空间不是完全现实的。

使用原则：

- 小面积使用。
- 出现在关键路径或异常细节上。
- 与灰、暗黄、黄昏橙形成反差。
- 不做成科幻霓虹城市风。
- 不覆盖千禧年旧屋、黄昏、中式梦核的主体氛围。

### 3.2 情绪曲线

| 阶段 | 情绪 | 说明 |
|---|---|---|
| 房间 B | 迷糊、陌生 | 玩家不知道自己为什么在这里 |
| 纸条 | 被提醒、轻微不安 | 文本很短，但像有人提前知道玩家会来 |
| 门 B | 第一次推进 | 玩家离开第一个封闭空间 |
| 房间 A | 发现出口但被拦住 | 门 A 上锁，形成小目标 |
| 柜子 | 找到办法 | 柜子里有钥匙，但钥匙来历不明 |
| 门 A | 离开前的异常感 | 门能打开，但外面不一定正常 |
| 户外 | 释放 + 悬念 | 玩家离开小屋，但天色不对 |

## 4. 文本替换总表

### 4.1 功能版文本

| 对象 | 功能版文本 |
|---|---|
| 纸条 X | 纸条上写着：离开房间。 |
| 门 B 未读纸条 | 门像是能打开，但你还没有想好要去哪。 |
| 门 B 打开 | 门打开了。 |
| 柜子第一次互动 | 柜子里有一把钥匙。 |
| 柜子重复互动 | 柜子是空的。 |
| 门 A 无钥匙 | 门上锁了。 |
| 门 A 有钥匙 | 试着用刚才的钥匙，结果门打开了。 |
| 终点 | Demo 结束。 |

### 4.2 氛围版文本方案 A：克制现实版

这一版更适合第一轮替换，文本短，容易实现。

| 对象 | 氛围版文本 |
|---|---|
| 纸条 X | 纸条上写着：别待在这个房间。 |
| 门 B 未读纸条 | 门没有锁，但你突然不太想碰它。 |
| 门 B 打开 | 门轴响了一下。 |
| 柜子第一次互动 | 柜子底下压着一把小钥匙。 |
| 柜子重复互动 | 柜子里只剩一层灰。 |
| 门 A 无钥匙 | 门从另一边锁住了。 |
| 门 A 有钥匙 | 钥匙转了一圈，门开了。 |
| 终点 | 你走出了屋子。外面的天色不太对。 |

### 4.3 氛围版文本方案 B：更接近《千禧年战纪》

这一版会稍微接入儿童组织和“放学后”的感觉，适合 v0.2。

| 对象 | 氛围版文本 |
|---|---|
| 纸条 X | 纸条上写着：放学后，别留在屋里。 |
| 门 B 未读纸条 | 门把手很凉。你想起纸条可能还没看完。 |
| 门 B 打开 | 门开了。外面的空气像刚下过雨。 |
| 柜子第一次互动 | 柜子里有一把小钥匙，下面压着半张圆牌。 |
| 柜子重复互动 | 柜子空了。那股樟脑丸味还在。 |
| 门 A 无钥匙 | 门锁着。门缝里有一点黄昏的光。 |
| 门 A 有钥匙 | 你试着用刚才的钥匙。门开了，光一下子挤了进来。 |
| 终点 | 你走出了屋子。远处好像有人在喊你的名字。 |

### 4.4 推荐采用

第一轮功能验收后，建议先采用 **方案 A：克制现实版**。

原因：

- 不会过早引入“放学后”等正式世界观元素。
- 文本短，容易落地。
- 氛围足够，但不会盖过功能测试。

等 v0.2 再切换或扩展到方案 B。

## 5. 对象命名替换

| 当前名称 | 氛围版名称 | 说明 |
|---|---|---|
| 纸条 X | 发黄纸条 | 更具体，更有年代感 |
| 门 B | 里屋门 | 从房间 B 通向房间 A |
| 房间 A | 外间 | 靠近出口的房间 |
| 房间 B | 里间 | 玩家出生房间 |
| 柜子 | 旧木柜 | 强化视觉印象 |
| 门 A | 出口门 | 通往户外 |
| 户外终点 | 门外黄昏 | 更接近梦核氛围 |

## 6. 场景氛围设计

### 6.1 房间 B：出生房间 / 里间

目标感受：

> 玩家醒来或出现的地方。不是危险现场，但明显不该久待。

建议元素：

- 灰水泥地
- 旧木桌
- 墙角废纸
- 半扇脏窗
- 发黄纸条
- 纸条背面有一小道荧光绿笔迹
- 墙上旧报纸
- 墙角有几处不自然的荧光绿粉笔痕、霉斑或旧贴纸残留
- 低矮墙体
- 光线偏暗，但纸条位置可见

不要做：

- 血迹
- 尸体
- 过强恐怖元素
- 大量杂物挡路

### 6.2 房间 A：外间 / 找钥匙空间

目标感受：

> 看起来离出口很近，但还差一步。

建议元素：

- 出口门明显可见
- 旧木柜放在侧面
- 地上有拖痕
- 墙面有水渍
- 门缝透出黄昏光
- 门锁边缘可以有一点微弱荧光绿反光
- 柜子抽屉缝隙里可以露出一点荧光绿
- 柜子颜色与墙体区分

关键设计：

- 玩家最好先看到门 A，再发现门上锁。
- 柜子应该在房间 A 内可见，但不是直接贴在门 A 旁边。
- “上锁 -> 找柜子 -> 拿钥匙 -> 回门口”的路线要成立。

### 6.3 户外区域：门外黄昏

目标感受：

> 玩家走出来了，但外面并不完全正常。

建议元素：

- 黄昏色光线
- 远处模糊围墙
- 低多边形树影
- 空地
- 一条看不清通向哪里的路
- 终点区域可以是一块亮一点的地面
- 终点区域边缘可出现非常淡的荧光绿轮廓
- 远处墙根或路面裂缝里可以有少量荧光绿反光

不要做：

- 真正开放大地图
- 新 NPC
- 新任务
- 新可互动物件

## 7. 视觉参考方向

### 7.1 低多边形风格

第一轮氛围版可以只做简单替换：

| 对象 | 白盒版 | 氛围版 |
|---|---|---|
| 墙体 | 深灰立方体 | 发黄墙面、低清贴图 |
| 地面 | 灰色平面 | 水泥地或脏木板 |
| 门 | 彩色方块 | 掉漆旧木门 |
| 柜子 | 棕色方块 | 老式木柜 |
| 纸条 | 白色小平面 | 发黄纸片 |
| 户外终点 | 绿色区域 | 黄昏光照区域，边缘带少量荧光绿异常色 |

### 7.2 色彩建议

| 区域 | 主色 | 辅色 | 氛围 |
|---|---|---|---|
| 房间 B | 灰、暗黄 | 脏白、少量荧光绿 | 闷、旧，出现轻微异常 |
| 房间 A | 灰褐、木色 | 黄昏橙、少量荧光绿 | 接近出口，但门和柜子有异常感 |
| 户外 | 橙黄、灰蓝 | 深绿、荧光绿边缘光 | 释放但不安心 |

### 7.3 荧光绿使用规范

荧光绿是“梦核异常”的提示色，不是主色。

推荐使用位置：

| 位置 | 用法 |
|---|---|
| 纸条背面 | 一小道荧光绿笔迹或符号 |
| 门锁 | 锁孔边缘轻微反光 |
| 柜子缝隙 | 露出一点不自然的绿光 |
| 墙角 | 少量荧光绿粉笔痕、霉斑或旧贴纸残留 |
| 户外终点 | 终点区域边缘的淡绿色轮廓 |
| 路面裂缝 | 极少量荧光绿反光 |

避免使用方式：

- 不要整面墙刷成荧光绿。
- 不要把房间照成霓虹夜店。
- 不要让荧光绿压过黄昏橙和旧屋灰黄。
- 不要让它变成科幻 UI 色。
- 不要每个物体都加绿光，只有关键物件或异常位置出现。

推荐比例：

| 色彩类型 | 建议占比 |
|---|---:|
| 灰、暗黄、木色、脏白等现实旧屋色 | 70% |
| 黄昏橙、灰蓝等环境光色 | 20% |
| 荧光绿异常点缀 | 5% 到 10% |

避免：

- 纯黑恐怖屋
- 大面积高饱和霓虹
- 过度血腥红色
- 现代写实高清材质

## 8. 可选环境调查文本

这些不是第一版必须实现的互动对象。它们是 v0.2 可以加的氛围调查点。

### 8.1 旧桌子

```text
桌面上有几道铅笔划痕。
其中一道像是有人用力写过“回家”。
```

### 8.2 脏窗户

```text
窗户外面是灰的。
你看不清是天快黑了，还是玻璃太脏。
```

### 8.3 墙上旧报纸

```text
报纸已经发黄。
日期被撕掉了，只剩下一个星期三。
```

### 8.4 地上的圆牌

```text
地上有半张圆牌。
边缘被磨得很亮，像是被很多人捏过。
```

### 8.5 门外

```text
外面有风。
但屋子里的灰没有动。
```

### 8.6 荧光绿粉笔痕

```text
墙角有一道很亮的绿色粉笔痕。
它不像粉笔，更像是什么东西在黑暗里自己亮着。
```

## 9. 最小演出建议

功能跑通后，可以加入极少量演出，但不要加复杂过场。

| 触发点 | 演出建议 | 是否必须 |
|---|---|---|
| 阅读纸条 | 文本框出现时镜头不动，停顿 0.2 秒 | 可选 |
| 打开门 B | 门消失或旋转打开 | 建议 |
| 搜索柜子 | 柜子轻微变色或打开 | 可选 |
| 打开门 A | 门打开后户外光变亮一点 | 可选 |
| 到达终点 | 屏幕稍微淡出，显示终点文本 | 建议 |

## 10. 音效建议

第一版无需音效。v0.2 如果加入，可以只加 4 个。

| 场景 | 音效 |
|---|---|
| 按 E 互动 | 轻微确认声 |
| 门打开 | 老木门轴声 |
| 搜索柜子 | 木柜摩擦声 |
| 到达户外 | 远处风声或放学铃余音 |

音效风格要克制，不做恐怖跳吓。

## 11. v0.2 扩展方向

如果功能版通过验收，v0.2 可以选择以下小扩展：

| 扩展 | 内容 | 风险 |
|---|---|---|
| 文本替换 | 使用方案 A 或方案 B 文本 | 低 |
| 氛围美术 | 替换墙、门、柜子、纸条材质 | 中 |
| 环境调查点 | 加 2 到 3 个可调查物 | 中 |
| 终点演出 | 到户外后淡出并显示悬念文本 | 低 |
| 简单音效 | 门、柜子、风声 | 低 |

不建议 v0.2 加：

- 战斗
- NPC
- 追逐
- 背包
- 复杂剧情
- 大地图

## 12. 给实现者的切换说明

当功能版通过 [[09 最小Demo测试验收表]] 后，可以向实现者交付本文件，并说明：

1. 不改主流程。
2. 不新增系统。
3. 只替换文本、显示名、基础模型、颜色、光照和少量演出。
4. 先采用“方案 A：克制现实版”文本。
5. 所有新增氛围内容不能影响原有验收表中的 P0 流程。
