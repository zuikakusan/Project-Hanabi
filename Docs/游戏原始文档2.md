# 《千禧年战纪》
# 1. 项目概述

## 1.1 游戏名称

《千禧年战纪》

## 1.2 游戏类型

第三人称射击游戏 TPS + 角色扮演 RPG + 日常模拟 + 小游戏合集 + 城市秘境探索。

## 1.3 核心概念

《千禧年战纪》是一款以千禧年前后中国小学生童年经验为核心的低多边形梦核 RPG/TPS 游戏。

玩家将扮演不同背景、不同区域、不同性别与年龄的孩子，在学校、家、街区、农村、城中村等区域中生活、玩耍、结识同伴、收集传闻，并在黄昏之后进入被称为“鬼屋”“秘境”“怪地方”的异常区域，使用爆竹、烟花、玩具枪、木棍等儿童物品与神秘敌人战斗，逐步探寻世界真相。

## 1.4 一句话介绍

在千禧年前后的中国城市里，一群小学生用爆竹、玩具、传闻和儿童游戏，对抗黄昏之后显露出来的另一个世界。

## 1.5 核心体验关键词

- 中国千禧年童年
    
- 小学生日常
    
- 学校与家庭
    
- 小卖部
    
- 爆竹与烟花
    
- 儿童组织
    
- 神秘街道
    
- 鬼屋与秘境
    
- 低多边形 PS2 风格
    
- 中式梦核
    
- 怀旧温馨
    
- 探索紧张
    
- 文本 RPG
    
- 第三人称射击
    
- 小游戏合集

# 2. 游戏定位

## 2.1 目标风格

本作希望以梦幻化、神秘化、游戏化的方式还原千禧年前后中国小学生的生活经验。

参考方向：

- 《女神异闻录5》：日常生活 + 异世界探索 + 角色关系 + 时间推进。
    
- 《放学后少年》：童年生活、小学生视角、小游戏与怀旧感。
    
- 《孤胆枪手 / Alien Shooter》：俯视/第三人称射击、怪物围攻、爽快战斗。
    
- 《极乐迪斯科》：文本深度、对话推进、内心能力、地图调查。
    
- PS2 时代 Early 2000s Low-Poly Game Style：低多边形、低清贴图、复古 3D 游戏质感。
    

## 2.2 不做什么

为了避免项目范围过大，当前阶段暂时不优先做：

- 完整开放世界。
    
- 多主角完整剧本。
    
- 大规模城市无缝地图。
    
- 复杂联网系统。
    
- 复杂战术射击。
    
- 真实暴力表现。
    
- 写实未成年人伤害表现。
    

当前更适合做成：

- 箱庭式开放街区。
    
- 章节制多主角。
    
- 单人 RPG/TPS。
    
- 日常区域 + 秘境区域双层结构。
    
- 小地图高密度互动。
    
- 低成本但风格明确的低多边形美术。

## 3.2 支柱二：儿童游戏与小学生江湖

游戏中包含大量中国小学生玩过的民间小游戏。

已确定或候选小游戏：

- 小刀一把
    
- 弹弹珠
    
- 圆牌 / 拍牌
    
- 打纸片
    
- 跳房子
    
- 躲猫猫
    
- 斗鸡
    
- 木头人
    
- 扔沙包

1. 推进 NPC 关系。
    
2. 作为任务的一部分。
    
3. 训练核心战斗能力。
    
4. 提供道具、称号、情报。
    
5. 表达儿童社会中的等级、面子、联盟和冲突。
    

## 3.3 支柱三：黄昏之后的城市秘境

游戏世界表面上是普通城市、学校和家庭。  
但某些区域并不完全符合课本、电视和大人所说的“科学世界”。

孩子们把这些区域称作：

- 鬼屋
    
- 怪地方
    
- 秘境
    
- 黑门后面
    
- 黄昏街道
    
- 不该存在的教室
    
- 旧地图之外的地方
    

秘境是战斗与探索的主要区域，基调从怀旧温馨转为紧张、刺激、梦核、压迫。

## 3.4 支柱四：爆竹 / 烟花 / 玩具武器战斗

玩家使用儿童物品进行战斗，包括：

- 窜天猴
    
- 鞭炮

# 4. 游戏基础循环

## 4.1 大循环

游戏的核心循环为：

1. 日常生活
    
2. 与 NPC 对话
    
3. 收集传闻 / 接受委托
    
4. 参与儿童小游戏
    
5. 获得道具、关系、情报
    
6. 找到秘境入口
    
7. 进入秘境探索
    
8. 使用爆竹 / 玩具武器战斗
    
9. 获得真相碎片 / 新能力 / 新地图线索
    
10. 回到日常，影响学校、家庭和儿童组织关系
    

## 4.2 单日循环

建议采用类似 P5 的时间段结构：

|时间段|可进行内容|
|---|---|
|早晨|上学路、家庭对话、偶遇同学|
|上午|上课、小纸条、老师点名、课堂事件|
|课间|小游戏、交换道具、打听传闻|
|午休|学校探索、秘密基地会议|
|放学后|街区探索、小卖部、小游戏挑战|
|黄昏|秘境入口出现，进入探索与战斗|
|夜晚|回家、家庭剧情、整理背包、写日记|

## 4.3 场景循环

每个区域建议包含：

1. 普通状态：白天、现实、日常。
    
2. 异常状态：黄昏、梦核、秘境入口出现。
    
3. 秘境状态：战斗、解谜、异常空间。
    
4. 回归状态：回到现实，但某些 NPC、道具或地图状态发生变化。

# 6. 世界观与组织设定

## 6.1 世界基本设定

大人、电视、课本和学校都宣称世界是科学、有序、可解释的。  
但孩子们逐渐发现，城市中存在许多无法被大人解释的区域。

这些地方通常在黄昏、放学后、雨天、停电时、假期、考试前后出现异常。

## 6.2 儿童神秘组织

游戏存在一个由当地孩子自发组成的神秘组织。

该组织在大的层面上只是一个共同身份，类似“冒险者协会”。  
不同区域有不同的小组织，彼此之间可以合作、竞争、敌对、交易。

组织共同目标：

- 探寻世界真相。
    
- 调查鬼屋和秘境。
    
- 收集异常道具。
    
- 记录大人看不见的东西。
    
- 守住孩子们自己的秘密世界。
    

## 6.3 组织规则候选

- 会员必须有代号。
    
- 不能向大人泄密。
    
- 新人必须完成胆量测试。
    
- 每个区域的小组织拥有自己的徽章。
    
- 道具可以交易，但不能赖账。
    
- 战利品需要按照约定分配。
    
- 进入秘境前必须报备。
    
- 背叛者会被贴上羞辱性称号。
    
- 最强者不一定最会打，而是最敢探索未知。

# 7. 角色设计方向

## 7.1 多主角结构

游戏计划拥有多个主角。  
不同主角初始区域不同，拥有不同背景故事、人物设定、性别和年龄。  
所有主角年龄均小于十四岁。

共同点：

- 都是孩子。
    
- 都生活在同一城市范围内。
    
- 都接触到儿童神秘组织。
    
- 都喜欢或擅长使用爆竹 / 烟花 / 玩具道具。
    
- 都逐渐发现现实世界背后的异常。
    

## 7.2 主角类型候选

### 主角 A：老城区小学男孩

区域：老城区、家属院、小学、小卖部。  
特点：胆大、爱逞强、熟悉爆竹。  
初始武器：擦炮、摔炮。  
主题：面子、朋友、谎言、逞强。

### 主角 B：农村女孩

区域：乡村小学、田埂、河边、废井、祠堂。  
特点：动手能力强，方向感好。  
初始武器：木棍、魔术棒。  
主题：孤独、传闻、家庭、野外恐惧。

### 主角 C：城中村孩子

区域：出租屋、批发市场、网吧、工地。  
特点：擅长交易、跑得快、消息灵通。  
初始武器：玩具枪、窜天猴。  
主题：流动、身份、边缘感。

### 主角 D：重点小学学生

区域：新城区、培训班、书店、少年宫。  
特点：成绩好、观察力强、战斗力弱。  
初始武器：低级玩具，但拥有高“幻想力”。  
主题：规训、成绩、压抑、逃离。

## 7.3 角色属性

建议采用儿童化属性，而不是传统力量、敏捷、智力。

候选属性：

|属性|作用|
|---|---|
|胆量|影响进入鬼屋、面对敌人、特殊对话|
|机灵|影响解谜、识破谎言、发现线索|
|面子|影响儿童社会地位、小游戏收益|
|幻想力|影响看见秘境、理解异常、特殊技能|
|手巧|影响道具制作、武器改造、拆解机关|
|跑得快|影响移动、闪避、逃跑|
|记仇|影响特殊对话、复仇任务、关系变化|
|讲义气|影响队友协作、组织评价|
|作业压力|影响夜晚行动、家庭限制|

# 8. 核心战斗系统

## 8.1 战斗类型

第三人称射击 TPS。  
推荐镜头为第三人称俯视角 / 半俯视角 / 斜 45 度视角。

参考：

- 《孤胆枪手》：大量敌人、火力压制、紧张刺激。
    
- PS2 低多边形动作游戏：简洁动作、强反馈。
    
- 儿童物品幻想化武器：爆竹、玩具枪、木棍。
    

## 8.2 战斗场景

战斗主要发生在秘境区域，例如：

- 废弃教室
    
- 鬼屋
    
- 废弃工厂
    
- 学校后巷
    
- 没有尽头的厕所
    
- 黄昏操场
    
- 小卖部后门
    
- 课本插图世界
    
- 城中村废楼
    
- 农村废井
    
- 旧祠堂
    

## 8.3 武器设计原则

武器不应只区分伤害，而应区分：

|参数|说明|
|---|---|
|伤害|命中后的基础输出|
|范围|爆炸或攻击半径|
|噪音|是否吸引敌人|
|引线时间|点燃到生效的延迟|
|弹道|直线、抛物线、持续发射等|
|暴露度|使用后是否暴露玩家位置|
|恐吓值|是否让敌人退缩|
|烟雾值|是否遮挡视线|
|稳定性|是否会偏移、哑火、反弹|
|稀有度|获取与补给难度|

## 8.4 初始四种爆竹武器

### 擦炮

类型：投掷 / AOE  
弹道：抛物线投掷  
效果：延迟后中范围爆炸  
伤害：中等  
特点：需要预判敌人位置  
缺点：有引线时间，可能被敌人躲开

### 摔炮

类型：投掷 / AOE  
弹道：抛物线投掷  
效果：落地小范围爆炸  
伤害：低  
特点：安静、即时、适合潜行  
缺点：范围小，伤害低

### 魔术棒

类型：持续射击  
弹道：从顶端连续发射小弹药  
效果：点燃后，每 1 秒可发射一枚小弹药  
持续时间：约 1 分钟  
伤害：小伤害，多段命中  
特点：适合持续压制  
缺点：单发伤害低，需要保持站位

### 窜天猴

类型：直线远程 / 爆炸  
弹道：沿瞄准方向直线飞出  
效果：命中后爆炸  
伤害：高  
特点：远距离、高爆发  
缺点：声音大，容易暴露位置，可能偏航

## 8.5 其他武器候选

|武器|定位|
|---|---|
|玩具枪|稳定远程，低伤害|
|木棍|近战，节省资源|
|双节棍|连击近战，高风险|
|仿真军刀|仪式化近战道具，建议避免写实表现|
|弹弓|精准射击，可装不同弹丸|
|激光笔|标记敌人、吸引注意|
|鞭炮串|大范围连续爆炸|
|烟雾弹式烟花|遮挡视线、逃跑|
|旋转陀螺烟花|地面范围控制|

## 8.6 战斗表现注意事项

由于主角均为儿童，游戏应避免写实呈现未成年人之间的严重伤害。

建议在程序与文本中使用以下表达：

- 出局
    
- 击退
    
- 打散
    
- 被梦境吞没
    
- 火花熄灭
    
- 回到现实
    
- 被判负
    
- 失去勇气值
    

避免将儿童小游戏或儿童之间的对抗写成真实致命暴力。

# 9. 小游戏系统

## 9.1 小游戏定位

小游戏是游戏的重要组成部分。  
小游戏不是纯娱乐附加内容，而是连接日常、社交、道具、成长、战斗的中间系统。

小游戏功能：

- 获得道具
    
- 提升属性
    
- 解锁 NPC 好感
    
- 赢得地盘
    
- 获取传闻
    
- 触发支线
    
- 训练战斗技巧
    
- 建立儿童社会规则
    

## 9.2 小游戏一：小刀一把

### 9.2.1 设计定位

“小刀一把”是一个儿童规则改造的对抗小游戏。  
它表现儿童之间的地盘、面子、等级、翻盘和装备夺取逻辑。

建议将其表现为抽象化、幻想化规则游戏，而不是写实伤害游戏。

### 9.2.2 基础设定

两个玩家进行对抗。  
每个玩家初始在自己的“家”中。  
初始等级为 1。  
通过猜拳获得行动权。  
赢得行动权后可以推进、拿武器、攻击、抢夺、撤退等。

### 9.2.3 基础状态

|状态|说明|
|---|---|
|等级|当前小游戏等级|
|位置|自己家 / 中间地带 / 对方家|
|武器|无 / 一级小刀 / 高级道具|
|士气|影响抢夺和防守|
|胆量|影响深入对方家时的稳定性|
|面子|影响胜负后的社交结果|
|主场优势|对方进入自己家后可获得反击机会|

### 9.2.4 回合流程

每回合流程：

1. 双方进行石头剪刀布。
    
2. 胜者获得一次行动机会。
    
3. 胜者选择行动。
    
4. 系统结算位置、武器、状态。
    
5. 判断是否有人出局。
    
6. 若无人出局，进入下一回合。
    

### 9.2.5 行动类型

|行动|条件|效果|
|---|---|---|
|拿武器|自己家或特定位置|获得一级武器|
|前进|未到对方家|向对方家推进一步|
|攻击|位于对方家且持有武器|使对方出局|
|抢夺|对方持有武器|尝试夺取对方武器|
|防守|任意位置|提高下一次防御成功率|
|逃回家|非自己家|返回自己家，保留部分状态|
|虚张声势|任意位置|干扰对方判断|
|叫人围观|任意位置|增加奖励，也增加失败惩罚|

### 9.2.6 翻盘机制

低等级玩家仍有翻盘机会。

翻盘机制包括：

- 主场优势：对方进入自己家时，抢夺成功率提高。
    
- 武器夺取：对方武器越强，被抢后的反转越大。
    
- 慌张值：深入敌方家时连续失败会累积慌张。
    
- 面子压力：等级越高，失败扣除越多面子。
    
- 特殊角色技能：不同孩子在小游戏中有不同特长。
    

### 9.2.7 胜利结果

胜利后可能获得：

- 等级提升
    
- 面子提升
    
- 新称号
    
- 道具
    
- 对方情报
    
- 进入儿童组织的资格
    
- 秘境线索
    
- 新小游戏挑战资格

# 13. 最小 Demo 设计

## 13.1 Demo 名称

《千禧年战纪：小刀一把》

## 13.2 Demo 目标

验证游戏最核心的三段式循环：

日常探索 → 小游戏 → 秘境战斗

## 13.3 Demo 范围

只做：

- 一个主角
    
- 一个学校区域
    
- 一个小卖部
    
- 一条学校后巷
    
- 一个神秘商店
    
- 一个小游戏：小刀一把
    
- 一个小型秘境
    
- 四种爆竹武器
    
- 三种敌人
    
- 一个简单 Boss
    
- 一条完整短任务
    

## 13.4 Demo 流程

1. 主角在家醒来。
    
2. 玩家可以在家中进行少量交互。
    
3. 出门上学。
    
4. 到达学校。
    
5. 课堂或课间收到纸条。
    
6. 放学后去小卖部门口。
    
7. 与 NPC 进行“小刀一把”小游戏。
    
8. 胜利后获得道具或情报。
    
9. 得知学校后巷存在神秘商店。
    
10. 玩家前往后巷。
    
11. 进入神秘街道。
    
12. 获得或购买四种爆竹武器。
    
13. 进入小型秘境。
    
14. 使用爆竹武器完成战斗。
    
15. 击败秘境敌人或找到秘密。
    
16. 回到现实。
    
17. 收到儿童组织邀请。
    
18. Demo 结束。
    

## 13.5 Demo 完成标准

Demo 视为完成，需要玩家可以完整体验：

- 移动
    
- 镜头控制
    
- 与物体交互
    
- 与 NPC 对话
    
- 触发任务
    
- 进行“小刀一把”
    
- 获得道具
    
- 打开背包
    
- 进入秘境
    
- 使用四种爆竹武器
    
- 击败敌人
    
- 触发结尾剧情

# 20. 当前项目核心总结

《千禧年战纪》的最大特色不是单独的 TPS、RPG、小游戏或怀旧题材，而是它们的组合：

中国千禧年小学生童年经验

- 儿童组织与小学生江湖
    
- 爆竹 / 烟花 / 玩具武器战斗
    
- 中式梦核秘境
    
- 低多边形 PS2 美术
    
- 文本 RPG 深度
    

当前最适合优先开发的 Demo 是：

《千禧年战纪：小刀一把》

Demo 核心目标：

让玩家在一个小型学校街区中，完成一次日常探索、一次儿童小游戏、一次神秘商店发现、一次爆竹武器战斗，并在结尾收到儿童组织邀请。

这个 Demo 如果成立，就证明游戏的核心循环成立：

日常生活 → 儿童游戏 → 道具成长 → 秘境探索 → 爆竹战斗 → 世界真相推进。
