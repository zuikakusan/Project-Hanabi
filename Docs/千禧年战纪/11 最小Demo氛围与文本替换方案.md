# 最小 Demo 氛围与文本替换方案

## 1. 文档目的

这份文档用于在“废弃小屋 Demo”功能跑通后，为它替换一层更接近《千禧年战纪》的氛围、文本和视觉方向。

当前实现者应优先完成 [[10 最小Demo实现交付说明]] 中的功能闭环。本方案不要求第一版立刻实现，但策划可以提前准备好，等功能通过验收后再进入 v0.2 氛围版。

## 2. 氛围版目标

功能版目标：

```text
纸条 -> 门 B -> 房间 A -> 柜子钥匙 -> 门 A -> 户外终点
```

氛围版目标：

```text
普通废弃小屋 -> 不太对劲的纸条 -> 被锁住的出口 -> 旧柜子里的钥匙 -> 门外异常的黄昏
```

氛围版不改变主流程，只改变：

- 文本语气
- 物件命名
- 场景细节
- 光线气质
- 终点提示
- 与《千禧年战纪》世界观的轻微连接

## 3. 氛围定位

### 3.1 关键词

- 千禧年初
- 小学生视角
- 废弃小屋
- 黄昏
- 灰尘
- 旧木门
- 生锈钥匙
- 发黄纸条
- 不该出现的出口
- 荧光绿
- 中式梦核
- 轻微不安，不做强恐怖

### 3.1.1 荧光绿定位：室内灯光与调色

荧光绿不是指把物体直接刷成亮绿色，也不是在场景里到处放绿色霓虹物件。

这里的“荧光绿”主要指室内灯光和后期调色倾向：色温偏黄，色调偏绿，整体更亮、更柔、更梦核。它应该像一种不自然的室内灯光和影像处理方式，覆盖在黑色墙体、水泥地、旧木柜这些固有色之上。

当前实装基准：

- 室内墙体使用黑色或近黑色，优先保证墙体与地面的强对比。
- 室内地面使用灰色水泥，可略带偏绿灰，但不能和墙体混在一起。
- 户外地面使用草绿色草地，不再沿用室内水泥地。
- 墙体适当加高，配合斜对角镜头增强小屋的体积感。
- 镜头采用斜对角等轴视角，从房间一个斜对角方向看进去，不使用正面轴向构图。
- 镜头只跟随玩家位置，不因左右或上下输入产生额外晃动。

使用原则：

- 室内整体色温偏黄。
- 室内整体色调偏绿。
- 增加曝光、高光和白色色阶，让画面更亮。
- 拉高黑色色阶，让暗部也变亮，不要死黑。
- 降低纹理和清晰度，形成柔光、低清晰、梦核质感。
- 增加鲜艳度和饱和度，让颜色更有生命力。
- 曲线上拉，整体提亮。
- 将突兀颜色调整到与主色调一致。
- 混色器方向：中间调偏青，阴影偏蓝绿，高光偏黄。
- 高光偏黄，用来模拟太阳光或黄昏光照。

### 3.2 情绪曲线

| 阶段 | 情绪 | 说明 |
|---|---|---|
| 房间 B | 迷糊、陌生 | 玩家不知道自己为什么在这里 |
| 纸条 | 被提醒、轻微不安 | 文本很短，但像有人提前知道玩家会来 |
| 门 B | 第一次推进 | 玩家离开第一个封闭空间 |
| 房间 A | 发现出口但被拦住 | 门 A 上锁，形成小目标 |
| 柜子 | 找到办法 | 柜子里有钥匙，但钥匙来历不明 |
| 门 A | 离开前的异常感 | 门能打开，但外面不一定正常 |
| 户外 | 释放 + 悬念 | 玩家离开小屋，但天色不对 |

## 4. 文本替换总表

### 4.1 功能版文本

| 对象 | 功能版文本 |
|---|---|
| 纸条 X | 纸条上写着：离开房间。 |
| 门 B 未读纸条 | 门像是能打开，但你还没有想好要去哪。 |
| 门 B 打开 | 门打开了。 |
| 柜子第一次互动 | 柜子里有一把钥匙。 |
| 柜子重复互动 | 柜子是空的。 |
| 门 A 无钥匙 | 门上锁了。 |
| 门 A 有钥匙 | 试着用刚才的钥匙，结果门打开了。 |
| 终点 | Demo 结束。 |

### 4.2 氛围版文本方案 A：克制现实版

这一版更适合第一轮替换，文本短，容易实现。

| 对象 | 氛围版文本 |
|---|---|
| 纸条 X | 纸条上写着：别待在这个房间。 |
| 门 B 未读纸条 | 门没有锁，但你突然不太想碰它。 |
| 门 B 打开 | 门轴响了一下。 |
| 柜子第一次互动 | 柜子底下压着一把小钥匙。 |
| 柜子重复互动 | 柜子里只剩一层灰。 |
| 门 A 无钥匙 | 门从另一边锁住了。 |
| 门 A 有钥匙 | 钥匙转了一圈，门开了。 |
| 终点 | 你走出了屋子。外面的天色不太对。 |

### 4.3 氛围版文本方案 B：更接近《千禧年战纪》

这一版会稍微接入儿童组织和“放学后”的感觉，适合 v0.2。

| 对象 | 氛围版文本 |
|---|---|
| 纸条 X | 纸条上写着：放学后，别留在屋里。 |
| 门 B 未读纸条 | 门把手很凉。你想起纸条可能还没看完。 |
| 门 B 打开 | 门开了。外面的空气像刚下过雨。 |
| 柜子第一次互动 | 柜子里有一把小钥匙，下面压着半张圆牌。 |
| 柜子重复互动 | 柜子空了。那股樟脑丸味还在。 |
| 门 A 无钥匙 | 门锁着。门缝里有一点黄昏的光。 |
| 门 A 有钥匙 | 你试着用刚才的钥匙。门开了，光一下子挤了进来。 |
| 终点 | 你走出了屋子。远处好像有人在喊你的名字。 |

### 4.4 推荐采用

第一轮功能验收后，建议先采用 **方案 A：克制现实版**。

原因：

- 不会过早引入“放学后”等正式世界观元素。
- 文本短，容易落地。
- 氛围足够，但不会盖过功能测试。

等 v0.2 再切换或扩展到方案 B。

## 6. 场景氛围设计

### 6.1 房间 B：出生房间 / 里间

目标感受：

> 玩家醒来或出现的地方。不是危险现场，但明显不该久待。

建议元素：

- 灰水泥地
- 旧木桌
- 墙角废纸
- 半扇脏窗
- 发黄纸条
- 墙上旧报纸
- 室内灯光带轻微黄绿色偏色
- 暗部被抬亮，不出现纯黑死角
- 黑色或近黑色墙体，墙体适当加高以形成体积感
- 光线偏暗，但纸条位置可见

不要做：

- 血迹
- 尸体
- 过强恐怖元素
- 大量杂物挡路

### 6.2 房间 A：外间 / 找钥匙空间

目标感受：

> 看起来离出口很近，但还差一步。

建议元素：

- 出口门明显可见
- 旧木柜放在侧面
- 灰水泥地
- 地上有拖痕
- 墙面有水渍
- 门缝透出黄昏光
- 室内阴影偏蓝绿，中间调略偏青
- 高光偏黄，像外部太阳光照进来
- 柜子颜色与墙体区分

关键设计：

- 玩家最好先看到门 A，再发现门上锁。
- 柜子应该在房间 A 内可见，但不是直接贴在门 A 旁边。
- “上锁 -> 找柜子 -> 拿钥匙 -> 回门口”的路线要成立。

### 6.3 户外区域：门外黄昏

目标感受：

> 玩家走出来了，但外面并不完全正常。

建议元素：

- 黄昏色光线
- 远处模糊围墙
- 低多边形树影
- 空地
- 户外地面改为草绿色草地，可以保留少量踩踏痕迹或泥土边缘
- 一条看不清通向哪里的路
- 终点区域可以是一块亮一点的地面
- 户外高光偏黄，模拟太阳光
- 阴影可略偏蓝绿，保持梦核层次

不要做：

- 真正开放大地图
- 新 NPC
- 新任务
- 新可互动物件

## 7. 视觉参考方向

### 7.0 色彩总原则：固有色 + 光照色 + 异常点缀色

当前氛围版的色彩逻辑需要明确区分三层：

| 层级 | 作用 | 示例 |
|---|---|---|
| 固有色 | 物体本身的颜色 | 黑色墙体、灰水泥地、草绿色户外地面、木色柜子、发黄纸张 |
| 光照色 | 环境光照在物体上的效果 | 高光偏黄、室内色调偏绿、阴影偏蓝绿 |
| 后期调色 | 统一画面气质 | 提曝光、抬黑位、降清晰度、加鲜艳度、曲线上拉 |

关键原则：

- 黄昏色是光照效果，不是物体材质颜色。
- 墙壁、地面、门、柜子、纸条都应该先有自己的固有色。
- 不要把所有物体都染成橙黄色。
- 黄昏橙应该像一层光照打在物体上，而不是把墙、地面、柜子统一改色。
- 荧光绿主要通过灯光、色调和阴影层次体现，不是把物体刷成绿色。

建议实现理解：

```text
黑色墙体 + 灰水泥地 + 草绿色户外 + 黄绿色室内柔光 + 蓝绿阴影 + 偏黄高光
```

而不是：

```text
所有墙体、地面、柜子都变成橙黄色或亮绿色
```

### 7.1 低多边形风格

第一轮氛围版可以只做简单替换：

| 对象 | 白盒版 | 氛围版 |
|---|---|---|
| 墙体 | 深灰立方体 | 黑色或近黑墙体，适当加高，优先形成清楚轮廓和体积感 |
| 室内地面 | 灰色平面 | 灰水泥地，保留水泥本色，可略偏绿灰，不做木地板，不整体染橙 |
| 户外地面 | 绿色区域 | 草绿色草地，和室内水泥地明确区分 |
| 门 | 彩色方块 | 掉漆旧木门，木色或灰褐色，门缝可有黄昏光 |
| 柜子 | 棕色方块 | 老式木柜，木色为主，受黄绿色室内柔光影响 |
| 纸条 | 白色小平面 | 发黄纸片或脏白纸片，受偏黄高光影响 |
| 户外终点 | 绿色或亮黄色区域 | 受黄昏光照照亮的出口区域，阴影略偏蓝绿 |

### 7.2 色彩建议

| 区域 | 物体固有色 | 光照色 | 异常点缀色 |
|---|---|---|---|
| 房间 B | 黑色墙体、灰水泥地、脏白纸张 | 室内黄绿色柔光，高光偏黄 | 阴影偏蓝绿，中间调偏青 |
| 房间 A | 黑色墙体、灰水泥地、木色柜子、灰褐门 | 门缝黄昏光，高光偏黄 | 阴影偏蓝绿，中间调偏青 |
| 户外 | 草绿色地面、深绿树影、灰色围墙 | 明显黄昏太阳光，高光偏黄 | 阴影略偏蓝绿 |

### 7.2.1 室内地面材质要求

室内地板方向明确为灰色水泥地。

要求：

- 地面是水泥地的感觉。
- 颜色以灰色、灰蓝、灰白、脏灰、偏绿灰为主。
- 可以有裂缝、灰尘、污渍、拖痕、粉笔灰。
- 可以被黄昏光照亮，但不能变成橙色地板。
- 可以受黄绿色室内灯光影响，但地面固有色仍然是灰水泥。
- 室内地面必须和黑色墙体拉开明度差异，保证玩家能分清地面和墙。

不要做：

- 木地板。
- 大面积橙黄色地板。
- 过于干净的现代瓷砖。
- 高反光金属地面。
- 室内大面积亮绿色地面。

### 7.2.2 灯光与后期调色要求

氛围版的梦核感主要来自灯光和后期，而不是物体贴图本身。

推荐方向：

| 调整项 | 方向 | 目的 |
|---|---|---|
| 色温 | 偏黄 | 模拟黄昏太阳光和旧室内灯 |
| 色调 | 偏绿 | 形成梦核式不自然感 |
| 曝光 | 增加 | 整体更亮，减少压抑死黑 |
| 高光 | 增加 | 让纸条、门缝光和地面受光处更亮 |
| 白色色阶 | 增加 | 强化纸张和受光地面亮部 |
| 黑色色阶 | 拉高 | 暗部也变亮，形成雾面柔光 |
| 纹理 | 降低 | 减少锐利材质感 |
| 清晰度 | 降低 | 形成柔光、梦核、低清晰感 |
| 鲜艳度 | 增加 | 增强整体色彩生命力 |
| 饱和度 | 适度增加 | 让色彩更明显，但避免廉价霓虹 |
| 曲线 | 整体上拉 | 提亮画面 |
| 中间调 | 偏青 | 让黑墙边缘和水泥地有青绿色层次 |
| 阴影 | 偏蓝绿 | 保留绿色层次，不让暗部发黑 |
| 高光 | 偏黄 | 模拟太阳光和黄昏照射 |

### 7.2.3 色彩修正要求

如果实装后画面“不好看”或“太橙”，优先检查以下问题：

- 墙体是否被直接做成橙黄色。
- 地面是否失去了灰水泥本色。
- 柜子是否被黄昏色吞掉，变得和墙面一样。
- 黄昏是否作为全屏滤镜过重。
- 绿色是否被理解成物体涂色，而不是灯光和调色。
- 暗部是否太黑，没有拉高黑色色阶。
- 画面是否太锐，没有柔光感。

正确方向：

- 墙体应是黑色或近黑色，形成清楚轮廓。
- 室内地面应该是灰水泥地，户外地面应该是草绿色草地。
- 柜子可以是木色或深棕色。
- 纸条可以是发黄白纸。
- 室内灯光色温偏黄、色调偏绿。
- 中间调偏青，阴影偏蓝绿，高光偏黄。
- 黄昏色只作为光线照在这些物体上。
- 暗部也要被抬亮，避免死黑。

### 7.3 绿色层次使用规范

绿色主要通过灯光、调色和明暗层次体现，不再优先理解为具体绿色物件。

推荐使用方式：

| 层级 | 用法 |
|---|---|
| 室内整体 | 色调偏绿，但不要把物体刷绿 |
| 中间调 | 偏青，影响黑墙边缘、水泥地和受光表面 |
| 阴影 | 偏蓝绿，暗部被抬亮 |
| 高光 | 偏黄，模拟太阳光 |
| 局部异常 | 可少量出现荧光绿粉笔痕或反光，但不是核心 |

避免使用方式：

- 不要整面墙刷成荧光绿。
- 不要把房间照成霓虹夜店。
- 不要让荧光绿压过黄昏橙和旧屋灰黄。
- 不要让它变成科幻 UI 色。
- 不要每个物体都加绿光。
- 不要让绿色变成廉价泛光污染。

推荐比例：

| 色彩类型 | 建议占比 |
|---|---:|
| 灰、暗黄、木色、脏白等现实旧屋色 | 70% |
| 黄昏黄、高光暖色、灰蓝环境色 | 20% |
| 青绿 / 蓝绿调色倾向 | 5% 到 10% |

避免：

- 纯黑恐怖屋
- 大面积高饱和霓虹
- 过度血腥红色
- 现代写实高清材质

## 9. 最小演出建议

功能跑通后，可以加入极少量演出，但不要加复杂过场。

| 触发点 | 演出建议 | 是否必须 |
|---|---|---|
| 阅读纸条 | 文本框出现时镜头不动，停顿 0.2 秒 | 可选 |
| 打开门 B | 门消失或旋转打开 | 建议 |
| 搜索柜子 | 柜子轻微变色或打开 | 可选 |
| 打开门 A | 门打开后户外光变亮一点 | 可选 |
| 到达终点 | 屏幕稍微淡出，显示终点文本 | 建议 |

## 10. 音效建议

第一版无需音效。v0.2 如果加入，可以只加 4 个。

| 场景 | 音效 |
|---|---|
| 按 E 互动 | 轻微确认声 |
| 门打开 | 老木门轴声 |
| 搜索柜子 | 木柜摩擦声 |
| 到达户外 | 远处风声或放学铃余音 |

音效风格要克制，不做恐怖跳吓。
